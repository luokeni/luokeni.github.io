js按需加载时间线

1、按需加载
1.按需加载HTML
2.按需加载图片
3.按需加载js
2、js时间线
1.创建Document对象，开始解析web页面，解析HTML元素和他们的文本内容后添加Element对象和Text节点到文档中。
这个阶段Document.readyState = “loading”。
2.遇到link外部css，创建线程加载，并继续解析文档。
3.遇到script外部js，并且没有设置async\defer ，浏览器加载，并阻塞，等待js加载完成并执行该脚本，然后继续解析文档
4.遇到script外部js，并且设置有async\defer 浏览器创建线程加载，并继续解析文档，
对于async属性的脚本，脚本加载完成后立即执行
5.遇到img标签等，先正常解析dom结构，然后浏览器异步加载src，并继续解析文档
6.当文档解析完成，document.readyState = “interactive”；
7.文档解析完成后，所有设置有defer的脚本会按照顺序执行。
8.当文档解析完成之后，document对象触发DOMContentLoaded事件，这也标志着程序执行从同步脚本执行阶段，转化为事件驱动阶段
9.当所有async的脚本加载完成并执行后，img等加载完成后，document.readyState = “complete” window对象触发load事件
​ 10.从此，页面以异步响应方式处理用户输入，网络事件等。

typora总结

typora总结

1.生成目录

[TOC]按回车 如果你想输入整篇文章的目录，只需在相应的位置键入[TOC]，目录便会呈现，并且支持点击跳转。

2.代码编辑块

三个点 + 回车键，三个点是左上角那个点（英文下）

设置代码块的颜色：https://jingyan.baidu.com/article/db55b609c807b04ba30a2fc7.html

代码区的右下角，在选择语言区域输入JavaScript

3.标题

在Typora中，只需按快捷键Ctrl+1，Ctrl+2，……，Ctrl+6
就可以依次输入一级到六级标题——标题的格式会立刻显现，并且在左端Outline窗口中出现，
在一篇文章的结构排好后，你当然可以在Outline窗口点击相应的标题进行跳转。

4.文字格式

加粗：ctrl+b
倾斜：ctrl+i
下划线：ctrl+u
居中对齐：

5.列表

无序列表：输入-之后输入空格
有序列表：输入数字+“.”之后输入空格 例如（2.）
任务列表：-[空格]空格 文字

6.引用

输入>之后输入空格

7.表格

表格：ctrl+t

8.选中一整行

ctrl+l

9.选中单词

ctrl+d

10.选中相同格式的文字

ctrl+e

11.跳转到文章开头

ctrl+home

12.跳转到文章结尾

ctrl+end

13.搜索

ctrl+f

14.替换

ctrl+h

15.删除线

alt+shift+5

16.插入图片

直接拖动到指定位置即可或者ctrl+shift+i

17.插入链接

ctrl+k

18.分割线

在新的一行输入3个-(中文字体下的)或者三个_（英文字体下的） 或者三个*
